# ADR 0003: Frontend con React + Vite + TypeScript

## Estado
Aceptado

## Contexto
Se requiere una SPA con buena DX, soporte TypeScript, HMR rápido y ecosistema rico en librerías UI.

## Decisión
React con Vite para un arranque veloz y build optimizada; TypeScript para robustez y autocompletado.

## Consecuencias
- Necesidad de configuraciones iniciales (ESLint, Tailwind, Zustand) pero reutilizables.
- Facilita code splitting y lazy loading.
- Amplio soporte de comunidad y librerías (charts, tablas, forms).
